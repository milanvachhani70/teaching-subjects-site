Maintenance

How To Add A New Subject Later

1

Create a new folder

Add a folder inside subjects/, for example subjects/physics101/.

2

Place the HTML file inside it

Keep the subject deck as index.html inside that folder, along with any images, CSS, or assets it needs.

3

Update one manifest entry

Add one object in subjects/manifest.js with title, description, and the subject path.

4

Push to GitHub

Once pushed to GitHub Pages, the new subject automatically appears on the homepage.
